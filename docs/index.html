Quick Start

Up and running in minutes

Build from source and run your first VisionPipe pipeline with just a few commands.

Clone the repository
Get the latest source from the official GitHub repository.
Install prerequisites
You need CMake 3.16+, a C++17 compiler, and OpenCV 4.x.
Build with CMake
Configure and compile. Enable optional features with CMake flags.
Write your pipeline
Create a .vsp file and run it with visionpipe run.
bash 
# Clone the official repository
git clone https://github.com/tlamngu/vision-pipe.git
cd vision-pipe

# Create build directory
mkdir build && cd build

# Configure (set OpenCV path)
cmake .. -DOpenCV_DIR=<path-to-opencv>

# Build in Release mode
cmake --build . --config Release

# Optional: enable ONNX Runtime support
cmake .. -DOpenCV_DIR=<path> \
         -DVISIONPIPE_WITH_ONNXRUNTIME=ON \
         -DOnnxRuntime_DIR=<path-to-onnxruntime>
vsp — hello.vsp 
# hello.vsp — Your first VisionPipe pipeline

pipeline main
    video_cap(0)
    draw_text("Hello from VisionPipe", 50, 50)
    imshow("Hello VisionPipe")
    wait_key(30)
end

exec_loop main
bash — run it 
visionpipe run hello.vsp

Key CMake Build Flags

VISIONPIPE_WITH_ONNXRUNTIME=ON
Enable ONNX Runtime backend for DNN inference
V4L2_NATIVE_BACKEND=ON
Direct V4L2 ioctl access on Linux (bypasses OpenCV)
VISIONPIPE_WITH_PYTHON=ON
Build pyvisionpipe Python bindings (requires pybind11)
VISIONPIPE_WITH_LIBCAMERA=ON
Enable libcamera support for Raspberry Pi cameras
DSL Guide

The VisionPipe DSL

A concise language for expressing vision workflows. Pipelines are named, composable, and support control flow, variables, and shared state.

pipeline — Define a named pipeline
A pipeline is a named block of vision operations. Pipelines can accept parameters and be called from other pipelines.
pipeline <name>(<params>)
<items>
end
exec_loop / exec_seq — Execute pipelines
exec_loop runs a pipeline in a continuous loop; exec_seq runs it once sequentially from another pipeline.
exec_loop main
exec_seq setup
exec_seq process(param1, param2)
global / global_cache — Shared frames
Pass frames between pipelines using named global variables. global_cache stores the current frame; use retrieves it.
flip("horizontal") -> global "frame"
use(global "frame")
global_cache("my_frame")
Variables & Control Flow
VisionPipe supports variables, arithmetic, and conditional branching with if, else, and break.
value = 42
if value == 42
draw_text("Match!", 10, 30)
end
let — Pipeline parameters
Pipelines can accept typed parameters using let. Pass values when invoking with exec_seq.
pipeline blur_frame(let size)
gaussian_blur(size)
end

exec_seq blur_frame(5)
vsp — full DSL example 
# Multi-pipeline vision script

# Pipeline with parameters
pipeline detect_edges(let low, let high)
    cvt_color("bgr2gray")
    gaussian_blur(5)
    canny(low, high)
    cvt_color("gray2bgr")
end

# Capture and store frame globally
pipeline capture
    video_cap(0)
    flip("horizontal") -> global "raw"
end

# Main loop: control flow + trackbar
pipeline main
    exec_seq capture

    trackbar_value("Low", "Ctrl", "lo")
    trackbar_value("High", "Ctrl", "hi")

    use(global "raw")

    if lo > 0
        exec_seq detect_edges(lo, hi)
    end

    imshow("Output")
    key = wait_key(1)
    if key == 27
        break
    end
end

# Entry point
exec_loop main
C++ — libvisionpipe 
#include "libvisionpipe/libvisionpipe.h"

int main() {
    vp::VisionPipeConfig cfg;
    cfg.script_path = "pipeline.vsp";

    vp::VisionPipe vp(cfg);
    auto session = vp.createSession();

    session->onFrame([](const vp::Frame& f) {
        // Zero-copy frame via shared memory
        cv::Mat img(f.height, f.width,
                    CV_8UC3, (void*)f.data);
        cv::imshow("frame", img);
    });

    vp.run();
    return 0;
}
Python — pyvisionpipe 
import pyvisionpipe as vp
import numpy as np

with vp.VisionPipe("pipeline.vsp") as pipe:
    session = pipe.create_session()

    # Callback mode
    def on_frame(frame: np.ndarray):
        print(f"Frame: {frame.shape}")

    session.on_frame(on_frame)
    pipe.run()
